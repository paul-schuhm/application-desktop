# Démo Dart - Poo basics, static, tooling

Une démo sur Dart, quelques éléments du langage et son tooling. Basée sur l'exercice 3.1

## Lancer la démo

~~~
dart run
~~~

## Générer la doc

~~~
dart doc
~~~

## Lancer l'analyse statique

~~~
dart analyze
~~~

## Lancer la suite de tests

~~~
dart test
~~~
